Italiano:

- Per poter caricare più file (pdf\image\etc) bisogna creare una cartella, zipparla, e caricare il file .zip, altrimenti è possibile caricare uno e un solo file.

English:

- 
In order to upload multiple files (pdf \ image \ etc) you need to create a folder, zip it, and load the .zip file, otherwise you can upload one and only one file.
